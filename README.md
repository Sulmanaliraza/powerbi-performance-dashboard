# Power BI Performance Dashboard

## Overview
This repository showcases a Power BI dashboard designed to monitor
service performance, KPIs, and SLA-style metrics to support
data-driven decision making.

## Business Use Case
Service organisations require clear visibility of performance trends
and compliance against KPIs to improve service quality and efficiency.

## Dashboard Features
- KPI summary (incidents, response time, resolution time)
- Performance trends over time
- SLA compliance by service line
- Incident distribution analysis

## Tools Used
- Power BI Desktop
- DAX (measures & calculations)
- CSV-based performance dataset

## Screenshots
See the `screenshots/` folder for dashboard visuals.

## Notes
This dashboard is built using sample data and is intended as a
portfolio demonstration of BI and performance reporting capability.
